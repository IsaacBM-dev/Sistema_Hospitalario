# 🏥 Sistema de Gestión Hospitalaria - Clínica Iberoamericana

Un sistema interactivo de consola desarrollado en Java para optimizar el registro de pacientes y la gestión de historias clínicas. Implementa arquitectura por capas y Programación Orientada a Objetos (POO). (Este proyecto es un sistema de prueba!).

## 🚀 Características Principales

* **Autenticación por Roles:** Sistema de login seguro con permisos específicos para Administrador, Médico y Recepcionista.
* **Gestión de Pacientes:** Registro de datos personales y búsqueda rápida mediante DNI.
* **Expediente Clínico:** Seguimiento de historias clínicas y resultados de análisis médicos.
* **Almacenamiento en Memoria:** Uso eficiente de colecciones (`ArrayList`) para la gestión temporal de datos.

## 🛠️ Tecnologías y Arquitectura

* **Lenguaje:** Java 
* **Entorno:** Eclipse IDE
* **Patrón de Diseño:** Separación en 3 capas (`main`, `model`, `system`).
* **Principios POO:** Herencia, Polimorfismo y Encapsulamiento.
